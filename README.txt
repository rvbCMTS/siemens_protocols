1) Install Access Database Engine on your local machine. To read the Microsoft
Access Database using pyodbc requires Access Database Engine installed on your
machine. Install the appropriate version from Microsoft.
https://www.microsoft.com/en-us/download/details.aspx?id=54920

2) Place your PEX database in the folder mdb_databases. It will run
all .mdb files in that folder.

3) Run main.py

4) Collect your .csv files in the folder mdb_databases.
